## 🔴 Critical Finding: 7 Policies Still Allow Anonymous Access

```sql
SELECT policyname, tablename FROM pg_policies 
WHERE schemaname='public' AND 'public'=ANY(roles);

-- Result:
-- mcp_public_read | companies
-- mcp_public_read | contacts
-- mcp_public_read | customers
-- mcp_public_read | insights
-- mcp_public_read | interactions
-- mcp_public_read | pipeline_stages
-- mcp_public_read | salespeople
```

**Risk:** Any request with anon key (no JWT) can READ all data from these tables.

**Remediation (after confirming no legitimate flow depends on this):**
```sql
DROP POLICY IF EXISTS mcp_public_read ON companies;
DROP POLICY IF EXISTS mcp_public_read ON contacts;
DROP POLICY IF EXISTS mcp_public_read ON customers;
DROP POLICY IF EXISTS mcp_public_read ON insights;
DROP POLICY IF EXISTS mcp_public_read ON interactions;
DROP POLICY IF EXISTS mcp_public_read ON pipeline_stages;
DROP POLICY IF EXISTS mcp_public_read ON salespeople;
```
